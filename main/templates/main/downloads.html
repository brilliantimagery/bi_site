Downloads

Windows

This was tested on 64 bit Windows 10, build 18362. Hopefully it'll work on other builds and versions but due to a lack of availability, that's yet to be known. Hopefully, in time, we'll be testing against more versions of Windows. If it doesn't work for you, we'd love to know!

  • Windows BrilliantImagery 0.1.0

Mac OS

Because of how this software is built, self-contained executable files can only be generated for the OS on which it's being built. We don't presently have easy access to any Mac OS powered machines so we can't practically build this. If you're up for helping make this happen please let us know! Also, other installation options are available so please check those out!

Linux

Since Adobe Lightroom isn't Linux compatible we aren't presently targeting Linux for a standalone executable. If there's demand, one can be made.
