Brilliant Imagery

Brillinat Imagery is a package and app for working with DNG images and image sequences. It can do things such as:

  • Parse DNG files and read their metadata.
  • Render DNG files.
  • Render lossless JPG files.
  • Smooth transitions between Adobe Lightroom edits in image sequences

It's built in a modular fashion so it can be used as a standalone app, extended, or used to extend the capabilities of something else.
